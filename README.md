<h2 id='sobre'>💻 Sobre o Projeto</h2>

<p>

Country Quiz - Esse desafio tem objetivo de criar um aplicativo de quiz sobre paises, onde todos os dados são fornecidos por uma API.
 
</p>
<!--final sobre -->

<h2 id='funcionalidade'>⚙️ Funcionalidades</h2>

- [X] O usuario consegue ver dois tipos de perguntas: uma cidade é a capital de um país, ou uma bandeira pertence a um país.
- [X] Pode selecionar a resposta;
- [X] Pode ver e a resposta está certa ou errada;
- [X] Quando a resposta está correta pode ir para a proxima pergunta;
- [X] Quando respondo incorretamente, posso ver meus resultados e tentar novamente;
- [X] Pode tentar novamente.
